# Crackdown on CHP (Yargı Kıskacında Demokrasi)

A comprehensive data journalism project examining the judicial pressure on Turkey's main opposition party.

## Overview

This project documents and visualizes:
- Legal cases against CHP members
- Timeline of political prosecutions
- Geographic distribution of cases

## Impact

The project provides transparent, data-driven documentation of political developments in Turkey.
