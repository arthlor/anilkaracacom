# Attack on Özgür Özel

A single-page interactive story documenting the attack against CHP Leader Özgür Özel.

## Overview

This project uses scrollytelling techniques to guide readers through the events, providing:
- Timeline of the incident
- Contextual information
- Visual storytelling elements

## Design Approach

The landing page format was chosen to create an immersive reading experience that combines narrative with data visualization.
